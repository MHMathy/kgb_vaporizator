# TD Outils

Permet de créer/modifier une image et de l'afficher dans une fenetre avec la bibliothèque SDL2

## Prérequis

Quelques informations pour pouvoir lancer le programme sans erreurs

### Installer la bibliothèque SDL2

Installer SDL2 sur linux se fais facilement depuis le terminal avec les commandes 

'''
hg clone https://hg.libsdl.org/SDL SDL
cd SDL
mkdir build
cd build
../configure
make
sudo make install
'''

Sur Windows il faudra télécharger la bibliothèque depuis le site et lancer l'installeur

## Comment Utiliser

### Compiler et Executer

Une fois dans le répertoire
'''
make
'''

### Fonctinnalités de affichage

*ZOOM* avec **T**

*DEZOOM* avec **G**

*Quitter* avec **Espace**
